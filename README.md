# misterios-de-eter
# Mistérios de Éter

Plataforma web para agendamento de consultas esotéricas (tarot, feitiços e trabalhos místicos), com cadastro/login de usuários e um painel administrativo completo para gestão de horários e agendamentos.

# Funcionalidades

- Cadastro e login de usuários
- Home pública com apresentação de serviços (leituras de tarot, trabalhos e feitiços)
- Dashboard do cliente para agendar horários, escolhendo serviço, data e horário disponíveis
- Dashboard do administrador com:
  - **Oráculo**: visão analítica de agendamentos (total, próximos, horários livres, gráfico de evolução mensal)
  - **Agenda**: gestão de disponibilidade de horários
- Sistema de status de agendamento (confirmado, concluído, cancelado) com indicação visual por cores

# Design

Identidade visual gótico-mística, com tipografia cursiva (Sacramento) para títulos e Cormorant Garamond para textos de interface, paleta de roxos e elementos decorativos (lua, bola de cristal, flores, corvo).

# 🛠️ Tecnologias

- **Frontend:** HTML5, CSS3 E JAVASCRIPT
- **Backend:** Node.js
- **Banco de dados:** SQL (MySQL)
